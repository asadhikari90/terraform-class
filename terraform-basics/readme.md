1) To initilize the emptry directory with supported plugin
# terraform init

2) To plan the execution 
# terraform plan 

3) To plan the execution and save it in a file
# terraform plan outputfile

4) To apply the infrastructure changed
# terraform apply 

5) To apply the infrastructure changes with the old save plan file
# terraform apply oldplanoutfile

6) To show the resources
# terraform show

7) To delete the insfrastructure
# terraform destroy 

# use the -target resource_type.name option with apply and destroy to work on a specific resource.
